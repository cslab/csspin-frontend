spin_frontend documentation
===========================

The ``spin_frontend`` plugin-package for cs.spin serves to provide the necessary
plugins and tools for the frontend development.

The sources can be found at https://code.contact.de/qs/spin/spin_frontend, while
the released versions are available at
https://packages.contact.de/tools/misc/spin-frontend.

``spin_frontend`` requires at least Python 3.9.
